== Description ==

Define up to 5 sources (URLs) in a metabox. Adds Source attribution metatag "original source" introduced by Google to single posts and page header. 
Adds your permalink to original source meta tag if external urls are not defined.

**Features**
_Add meta tag:_
* "original-source" - Google meta tag for Google news publishers and SEO

== Installation ==

1. Download, unzip and upload to your WordPress plugins directory
2. activate the plugin within you WordPress Administration
3. In a "add new" post or page, add up to 5 URLs to define original sources or let it blank to have your permalink defined as the original source
4. Done ! you are now a wordpress expert with Google compliant metatags


== Frequently Asked Questions ==

= What is "original source" meta tag ? =

original-source metatag was introduced by Google on November 2010 
(see also www.google.com/support/news_pub/bin/answer.py?answer=191283).
It indicates which URL(s) should be credited with breaking a story. 
Tags are standard HTML metatags, and should be added to the <head> element of the article page :
<meta name="syndication-source" content="*ttp://www.example.com/wire_story_1.html">

= What if i publish an article that consist entirely of original reporting =

Let the fields blank, the plugin will add automatically an original source metatag with your permalink 

= I want to define several sources =

Great, the plugin add a metabox and let you define up to 5 sources URL(s)
