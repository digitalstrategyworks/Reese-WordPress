== Description ==

HTML Javascript Adder allows to insert HTML, javascripts, shortcodes, advertisements, flash embed codes, plain text etc in the sidebar via UNLIMITED widgets with advanced targeting on posts and pages. No need to edit any file and it is 100% plugin based.

= Features =

* Used for adding HTML, Javascripts, Shortcodes, Flash embed codes and other related codes in the sidebar using widgets.
* Unlimited widget support.
* Can insert shortcodes also.
* Advanced settings for targeting widget in Posts and Pages. Seperate settings to Show/Hide a widget in a page eg: Can Show/hide a widget in home page, archive page etc.
* Available in 10 Languages.
* AW Quick Tag toolbar for editing codes in widget.
* 100% plugin based.

= New Features in version 3.x =

* Automatically add paragraph option.
* Can see the preview of the HTML. 
* Can show/hide the widget to Administrators (Helpful in hiding ads to admins).
* Can disable the widget in specific posts.
* Updated 10 languages.
* Revised widget form and code.

See [Change Log](changelog/) for more infomation

= What is the use ? =

You can use this plugin to insert anything in your sidebar. Some common suggestions are,

* Advertisements javascript codes (eg: Google adsense, Buysellads, adbrite etc)
* Flash items (eg: Youtube videos, JW Players, Slideshows, flash games made by you etc)
* Insert any shortcodes inside the widget.
* Link any files like CSS Stylesheets, Javascript files, libraries etc.
* Social sharing buttons javascript codes (eg: AddThis, shareThis, addtoany, tell-a-friend etc)
* Write any announcements, some navigation links etc.
* Add Facebook Like widget, Google Friend Connect widget etc.
* And more according to your needs.

== Installation ==

1. Extract the zipped file and upload the folder `html-javascript-adder` to to `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Go to the Widgets settings page.
1. Drag the "HTML Javascript Adder" from the "Available widgets" to your sidebar.
1. That's all.

= Usage = 

1. Just add a widget.
1. Enter the title and required content.
1. Check the boxes present below to show/hide the widget in specific pages.
1. Click save.

For usage, just see the screenshots. For detail usage and to use some inbuilt functions and features refer the [Documentation](http://www.aakashweb.com/wordpress-plugins/html-javascript-adder/) page in [Aakash Web](http://www.aakashweb.com/)

== Frequently Asked Questions ==

= What content should be entered in the Text/Javascript/HTML box =

You can enter any script like simple texts, javascripts and HTML Codes. See the description tab for some suggestions.

= I cannot see the preview of flash items =

This is a known issue. Preview will not be seen only when `<object>` tag is used. Works in all other places well.

= Can i enter javascripts/HTML in the "Title" box ? =

No please don't enter. Though, the widget accepts the code the output will be bad and it is not yet tested.

= What to do if i fall into trouble ? =

You will not fall into trouble, it is rare. If you got any trouble, just deactivate the plugin and activate it again.

For more FAQs please visit the [Plugin homepage](http://www.aakashweb.com/wordpress-plugins/html-javascript-adder/)
