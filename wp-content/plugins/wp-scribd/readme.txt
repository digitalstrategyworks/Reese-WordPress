== Installation ==

1. Upload contents of the ZIP file to your wp-content/plugins folder.
2. Activate the "WordPress Holidays" plugin
3. Go the the WordPress Holidays Option admin menu and check is it ok...

if you want upgrade manualy:

1. Deactivate plugin
2. Delete old files
3. Upload new files
